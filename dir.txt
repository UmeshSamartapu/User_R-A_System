Environment: 
Python 3

Build Command: 
pip install -r requirements.txt

Start Command: 
uvicorn app.main:app --host=0.0.0.0 --port=10000

Environment Variables:
MONGODB_URI: your Mongo URI
JWT_SECRET_KEY: a long secure key
JWT_ALGORITHM: HS256
JWT_EXPIRE_MINUTES: 60
